# Bbox

## Description
`Bbox(minX, minY, maxX, maxY)`

Returns a bounding box for a set of coordinates.

### Return Type
`Geometry`

## Parameters
| Parameter |              | Type(s)     | Description | `null` Behavior |
| ---       | ---          | ---      | ---         | --- |
| minX      | **required** | <code>Double</code>  | The minimum x coordinate.
