# Crear ambiente virtual
``` bash
python -m venv myenv
python -m venv nombre_entorno_virtual
```

# Activar ambiente virtual
``` bash
source myenv/bin/activate
```

``` bash
myenv\Scripts\activate
```

# Desactivar ambiente virtual
``` bash
deactivate
```


# Eliminar ambiente virtual
``` bash
rm -rf myenv
```

# Correr scripts
``` bash
python script.py
```
